# 有限会社Rispondere コーポレートサイト【デザイン強化版】

本案件は、指示書の内容を完全に実装し、さらにデザイン強化版として大幅にアップグレードしています。

## 🎨 デザイン強化版の特徴

### 1. CMS管理可能なデザインカラー
- メインカラー（赤→ネイビーなど自由に変更可能）
- サブカラー
- 背景色
- 文字色

**すべてHEX形式（#xxxxxx）でCMSから編集できます。**

### 2. バナー画像管理システム
川満さんが制作するバナー画像を、CMSから簡単に差し込めます。

- **トップページ**: ヒーローバナー 1枚
- **採用ページ**: 縦長バナー 0〜3枚（任意表示）
  - 上部バナー
  - ポイント後バナー
  - 給与後バナー
  - 最後のバナー

各バナーはON/OFF切替可能です。

### 3. デザイン要素
参考画像の構造をHTMLで完全再現：

- ✅ 大見出し（帯・ピル型）
- ✅ 重要数字を大きく（給与・年収）
- ✅ ポイントを3つ（01/02/03の番号付き）
- ✅ 仕事内容カード（角丸＋影）
- ✅ 勤務・福利厚生の枠
- ✅ 応募の不安を下げる締め

## 制作仕様

- **制作方式**: 純HTML / CSS / JavaScript
- **公開**: GitHub Pages
- **管理画面**: Decap CMS（旧Netlify CMS）
- **管理画面URL**: `/admin/`
- **認証**: GitHub OAuth

## セットアップ手順

### 1. GitHub Pages の設定

1. GitHubでリポジトリを作成
2. コードをプッシュ
3. リポジトリの Settings > Pages で以下を設定：
   - Source: `Deploy from a branch`
   - Branch: `main` / `/ (root)`
   - Save

### 2. Decap CMS の設定

`/admin/config.yml` の以下の行を編集：

```yaml
backend:
  name: github
  repo: OWNER/REPO  # ← 実際のリポジトリ名に変更（例: username/rispondere）
```

### 3. GitHub OAuth の設定（オプション）

Netlify Identity または GitHub OAuth App を設定することで、管理画面へのログインが可能になります。

簡易的には、GitHubから直接コンテンツファイルを編集することも可能です。

## 🎨 デザインカラーの変更方法

### CMS管理画面から変更

1. `/admin/` にアクセス
2. 「デザイン設定」→「カラー設定」を選択
3. HEX形式（例: `#ff0000`）で色を入力
4. 保存 → 公開

### 変更できるカラー

| カラー名 | 用途 | デフォルト値 |
|---------|------|-------------|
| メインカラー | ヒーロー、ボタン、見出し | `#d32f2f` (赤) |
| サブカラー | アクセント | `#f44336` (明るい赤) |
| 背景色 | ページ背景 | `#ffffff` (白) |
| 文字色 | 本文テキスト | `#333333` (濃いグレー) |

**ネイビーに変更したい場合**: メインカラーを `#2c3e50` に設定

## 🖼️ バナー画像の管理方法

### CMS管理画面から管理

1. `/admin/` にアクセス
2. 「デザイン設定」→「バナー画像管理」を選択
3. 各バナーの設定：
   - **表示する**: チェックを入れると表示される
   - **画像**: アップロードまたは既存画像を選択
4. 保存 → 公開

### バナー配置場所

#### トップページ
- **ヒーローバナー**: トップページの最上部（横長推奨）

#### 採用ページ
- **トップバナー**: ヒーロー直後（横長推奨）
- **バナー1**: 「この仕事のポイント」の後（縦長推奨）
- **バナー2**: 「給与・年収」の後（縦長推奨）
- **バナー3**: 最後（縦長推奨）

## 📋 採用ページの構成

デザイン強化版の採用ページは、以下の順序で構成されています：

1. **ヒーロー**（帯＋キャッチ）
2. **トップバナー**（任意）
3. **導入文**（安心・未経験OK）
4. **この仕事のポイント**（01〜03の番号付き）
5. **バナー1**（任意）
6. **給与・年収**（数字を大きく）
7. **給与の仕組み**（手当リスト）
8. **バナー2**（任意）
9. **仕事内容**（カード表示）
10. **勤務について**（枠付き）
11. **福利厚生**（枠付き）
12. **バナー3**（任意）
13. **応募について**（安心の締め）
14. **お問い合わせ導線**（ボタン＋メール）

すべてCMS管理画面から編集できます。

## 🎨 デザインのポイント

### 色味の変更が無料で可能
CSS変数で管理しているため、CMSから簡単に色を変更できます。
赤系→ネイビー系への変更も、HEXコードを変えるだけで完了します。

### バナー画像を自由に差し込み
川満さんが制作したバナー画像を、CMSから簡単にアップロード・配置できます。
ON/OFF切替で、表示・非表示も自由に制御できます。

### 「シンプルすぎ問題」を解決
- 1セクション＝1テーマで明確に区切り
- 見出しは帯（ピル型）で統一
- 数字は特大タイポで主役に
- ポイントは番号付きカードで分かりやすく
- 仕事内容はカード（角丸＋影）で並べる

## 注意事項

- フレームワーク・ビルドツール不使用
- 純HTML/CSS/JavaScriptのみで構成
- 管理画面からの編集内容は `/content/` 内のJSONファイルに保存
- GitHub Pages にデプロイ後は HTTPS で自動的に配信
