## 準備：必要なアカウントを作る

### ✅ 必要なアカウント（2つ）

1. **GitHubアカウント**（無料）
   - 役割：コードを保存する場所
   - URL：https://github.com/

2. **Netlifyアカウント**（無料）
   - 役割：サイトを公開する場所
   - URL：https://www.netlify.com/

---

### 📝 GitHubアカウントの作り方

**所要時間：3分**

1. https://github.com/ にアクセス
2. 右上の **「Sign up」** をクリック
3. メールアドレスを入力
4. パスワードを設定（英数字8文字以上）
5. ユーザー名を決める（例：rispondere-corp）
6. メール認証コードを入力
7. 完了！

> 💡 **ポイント**：ユーザー名は後で変更できないので、会社名などわかりやすいものにしましょう。

---

### 📝 Netlifyアカウントの作り方

**所要時間：1分**

1. https://www.netlify.com/ にアクセス
2. 右上の **「Sign up」** をクリック
3. **「Sign up with GitHub」** を選択
   - 先ほど作ったGitHubアカウントでログイン
4. 「Authorize Netlify」をクリック
5. 完了！

> 💡 **ポイント**：GitHubアカウントでログインすると、後の作業が簡単になります。

---

## ステップ1：GitHubにコードを保存する

**所要時間：5分**

### 1-1. GitHubで新しいリポジトリを作る

「リポジトリ」= プロジェクトを保存する場所（フォルダのようなもの）

1. GitHubにログインした状態で、右上の **「+」** マークをクリック
2. **「New repository」** を選択
3. 以下の項目を入力：

   | 項目 | 入力内容 | 説明 |
   |------|----------|------|
   | **Repository name** | `rispondere-website` | プロジェクトの名前（英数字のみ） |
   | **Description** | `有限会社Rispondereコーポレートサイト` | 説明（日本語OK） |
   | **公開設定** | 🔘 Public（公開）| 誰でも見られる（無料プラン） |
   | **Initialize** | ✅ Add a README file | READMEを追加（チェック） |

4. 一番下の **「Create repository」** をクリック
5. 完了！

---

### 1-2. ローカルのコードをGitHubにアップロードする

現在、あなたのパソコン（正確にはサンドボックス環境）にコードがあります。  
これをGitHubに送信（プッシュ）します。

> ⚠️ **注意**：この作業は、私（AI）が代行します。以下は参考情報です。

**コマンド（参考）：**
```bash
# GitHubリポジトリのURLを登録
git remote add origin https://github.com/あなたのユーザー名/rispondere-website.git

# コードをGitHubに送信
git branch -M main
git push -u origin main
```

## ステップ2：Netlifyでサイトを公開する

**所要時間：5分**

### 2-1. Netlifyにログイン

1. https://app.netlify.com/ にアクセス
2. GitHubアカウントでログイン

---

### 2-2. 新しいサイトを作成

1. ダッシュボードの **「Add new site」** をクリック
2. **「Import an existing project」** を選択
3. **「GitHub」** を選択
4. GitHubとの連携を許可（初回のみ）
5. リポジトリ一覧から **「rispondere-website」** を選択

---

### 2-3. ビルド設定

以下の設定画面が表示されます：

| 項目 | 設定内容 |
|------|----------|
| **Branch to deploy** | `main`（そのまま） |
| **Build command** | 空欄（何も入力しない） |
| **Publish directory** | `.`（ドットを入力） |

> 💡 **解説**：このサイトは純粋なHTML/CSS/JSなので、ビルドコマンドは不要です。

**「Deploy site」** をクリック！

---

### 2-4. デプロイ完了を待つ

- 数秒～1分ほどで「Published」と表示されます
- サイトのURLが自動生成されます
  - 例：`https://あなたのサイト名.netlify.app`

**このURLをクリックしてサイトを確認しましょう！**

🎉 **おめでとうございます！サイトが公開されました！**

---

## ステップ3：管理画面を有効化する

**所要時間：10分**

現在、サイトは公開されましたが、管理画面（`/admin/`）にはまだログインできません。  
管理画面を有効化するには、**GitHub OAuthの設定**が必要です。

---

### 3-1. Netlifyで「Identity」を有効化

1. Netlifyのダッシュボードで、あなたのサイトを選択
2. 上部メニューの **「Integrations」** をクリック
3. **「Identity」** の **「Enable Identity」** をクリック
4. 完了！

---

### 3-2. GitHub OAuthを有効化

1. 同じ画面で、下にスクロール
2. **「Settings and usage」** の **「Enable Git Gateway」** をクリック
3. 「GitHub OAuth App」の設定が自動的に行われます
4. 完了！

---

### 3-3. 管理画面にアクセス

1. ブラウザで以下のURLにアクセス：
   ```
   https://あなたのサイト名.netlify.app/admin/
   ```

2. **「Login with GitHub」** ボタンが表示されます
3. クリックしてGitHubアカウントでログイン
4. 「Authorize Netlify」をクリック

🎉 **管理画面にログイン成功！**

## 管理画面の使い方

管理画面にログインすると、以下の項目を編集できます：

### 📝 編集できる項目

#### 1️⃣ デザイン設定
- **サイトのカラー**（メイン色、サブ色、背景色、文字色）
- **バナー画像**（トップページ、採用ページ）

#### 2️⃣ ページ管理
- **トップページ**（メインコピー、本文）
- **事業内容ページ**（タイトル、本文）
- **働き方ページ**（タイトル、本文）
- **会社概要ページ**（会社情報）

#### 3️⃣ 採用ページ（2本立て）
- **業務サポート職**
  - ヒーローセクション
  - 3つのポイント
  - 給与情報
  - 1日の流れ
  - 仕事内容カード
  - 向いている人／向いていない人
- **デザイン職**（将来枠）
  - ステータス
  - 紹介文
  - 求めるスキル

#### 4️⃣ よくある質問（FAQ）
- 質問の追加・編集・削除
- 表示順序の変更

---

### ✏️ 編集の手順

1. 編集したい項目をクリック
2. テキストや画像を変更
3. 右上の **「Save」** をクリック
4. 上部の **「Publish」** → **「Publish now」** をクリック

> 💡 **ポイント**：Publishするまで、変更は公開されません。

数分後、サイトに変更が反映されます！

## 困ったときは

### ❓ よくあるトラブルと解決方法

#### 1. 管理画面にログインできない

**原因**：GitHub OAuthが正しく設定されていない

**解決方法**：
1. Netlifyのダッシュボード → **「Integrations」** → **「Identity」**
2. 「Enable Git Gateway」が有効になっているか確認
3. 無効なら、もう一度有効化

---

#### 2. 変更がサイトに反映されない

**原因**：Publishしていない、またはキャッシュ

**解決方法**：
1. 管理画面で **「Publish」** をクリックしたか確認
2. ブラウザで「スーパーリロード」を実行
   - **Windows**：`Ctrl + Shift + R`
   - **Mac**：`Command + Shift + R`

---

#### 3. 画像がアップロードできない

**原因**：画像サイズが大きすぎる

**解決方法**：
1. 画像を圧縮する（推奨サイズ：1MB以下）
2. オンラインツールを使う
   - https://tinypng.com/（無料）
   - https://squoosh.app/（無料）

---

#### 4. GitHubのリポジトリが見つからない

**原因**：NetlifyがGitHubにアクセスする権限がない

**解決方法**：
1. GitHub にログイン
2. 右上のアイコン → **「Settings」**
3. 左メニュー → **「Applications」** → **「Authorized OAuth Apps」**
4. **「Netlify」** を見つけて権限を確認
5. 権限がない場合は、Netlifyから再度連携
