# 有限会社Rispondere コーポレートサイト 管理画面編集マニュアル【デザイン強化版】

## 🎨 デザイン強化版の新機能

このバージョンでは、以下の新機能が追加されました：

1. **デザインカラーの変更**（赤→ネイビーなども可能）
2. **バナー画像の管理**（川満さん制作のバナーを自由に配置）
3. **採用ページの詳細管理**（すべての要素をCMSで編集）

---

## 1. 管理画面へのアクセス

### 管理画面URL
```
https://[あなたのドメイン]/admin/
```

### ログイン方法
1. 管理画面URLにアクセス
2. GitHubアカウントでログイン
3. リポジトリへのアクセス権限を承認

## 2. 🎨 デザインカラーの変更方法

### 手順
1. 管理画面にログイン
2. 左サイドバーから「デザイン設定」を選択
3. 「カラー設定」をクリック
4. 各カラーをHEX形式（例: `#ff0000`）で入力

### 変更できるカラー

| カラー名 | 用途 | デフォルト値 |
|---------|------|-------------|
| メインカラー | ヒーロー背景、ボタン、見出し | `#d32f2f` (赤) |
| サブカラー | アクセント色 | `#f44336` (明るい赤) |
| 背景色 | ページ全体の背景 | `#ffffff` (白) |
| 文字色 | 本文テキスト | `#333333` (濃いグレー) |

### 色の例

**ネイビー系に変更したい場合:**
- メインカラー: `#2c3e50`
- サブカラー: `#34495e`

**グリーン系に変更したい場合:**
- メインカラー: `#2e7d32`
- サブカラー: `#4caf50`

**ブルー系に変更したい場合:**
- メインカラー: `#1976d2`
- サブカラー: `#2196f3`

### 保存方法
1. すべてのカラーを入力
2. 右上の「Save」ボタンをクリック
3. 「Publish」を選択して公開
4. 数分後、サイトの色が変更されます

## 3. 🖼️ バナー画像の管理方法

### バナーの種類

#### トップページ
- **ヒーローバナー**: トップページ最上部（横長推奨）

#### 採用ページ
- **トップバナー**: ヒーロー直後（横長推奨）
- **バナー1**: 「この仕事のポイント」の後（縦長推奨）
- **バナー2**: 「給与・年収」の後（縦長推奨）
- **バナー3**: 最後（縦長推奨）

### バナー設定手順

1. 管理画面にログイン
2. 左サイドバーから「デザイン設定」を選択
3. 「バナー画像管理」をクリック
4. 設定したいバナーを選択

#### 各バナーの設定項目

**表示する**: 
- チェックを入れる → バナーが表示される
- チェックを外す → バナーが非表示になる

**画像**:
- 「Upload」タブから新しい画像をアップロード
- または「Choose an image」から既存の画像を選択

### バナー画像の推奨サイズ

- **横長バナー**（トップページ、採用トップ）: 幅1200px〜1600px
- **縦長バナー**（採用ページ内）: 幅600px〜800px

### 保存方法
1. すべてのバナーを設定
2. 右上の「Save」ボタンをクリック
3. 「Publish」を選択して公開

---

## 4. 📝 採用ページの編集方法

### 編集できる項目

#### ヒーローセクション
- バッジ（例: 「正社員募集」）
- メインタイトル
- サブタイトル

#### 導入文
- 未経験歓迎などの安心文

#### この仕事のポイント（3つ）
- ポイント1: タイトル・説明
- ポイント2: タイトル・説明
- ポイント3: タイトル・説明

#### 給与情報
- 年収例（数字のみ）例: `500〜600万円`
- 年収注記（※の説明）
- 月収例（数字のみ）例: `30〜40万円`
- 月収注記（※の説明）

#### 手当・給与の仕組み
- リスト形式で追加・削除可能
- 例: 「基本給 + 各種手当」「賞与（年2回）」

#### 仕事内容カード
- 各カードのタイトルと説明
- 最大6枚まで追加可能

#### 勤務について・福利厚生
- リスト形式で追加・削除可能

#### 応募について
- 応募の不安を下げる締めの文章

### 編集手順

1. 管理画面にログイン
2. 左サイドバーから「採用ページ」を選択
3. 「採用情報詳細」をクリック
4. 各セクションを編集
5. 右上の「Save」→「Publish」で公開

---

## 5. 基本的な編集項目

### ステップ1: ログイン
1. 管理画面URL（`/admin/`）にアクセス
2. 「Login with GitHub」をクリック
3. GitHubアカウントでログイン

### ステップ2: 編集したい項目を選択
1. 左サイドバーから編集したいカテゴリを選択
   - 「デザイン設定」← **NEW**
   - 「サイト設定」
   - 「ページ」
   - 「採用ページ」← **更新**
   - 「よくある質問」
2. 編集したい具体的な項目をクリック

### ステップ3: 内容を編集
1. テキストフィールドに新しい内容を入力
2. 改行を含む場合は、エンターキーで改行を入れる
3. 画像を変更する場合は、画像フィールドから選択またはアップロード

### ステップ4: 保存
1. 編集が完了したら、右上の「Save」ボタンをクリック
2. 「Publish」を選択して公開

### ステップ5: 確認
1. 数分後、実際のサイトにアクセスして変更を確認
2. GitHub Pagesの更新には1〜5分程度かかる場合があります

---

## 6. よくある質問（FAQ）の編集

### 新しいFAQを追加する
1. 左サイドバーから「よくある質問」を選択
2. 右上の「New よくある質問」をクリック
3. 質問と回答を入力
4. 表示順（数字が小さいほど上に表示）を設定
5. 「Save」→「Publish」で公開

### 既存のFAQを編集する
1. 左サイドバーから「よくある質問」を選択
2. 編集したいFAQをクリック
3. 内容を編集
4. 「Save」→「Publish」で公開

### FAQを削除する
1. 編集画面で「Delete entry」をクリック
2. 確認メッセージで「OK」を選択

---

## 7. 画像の編集・追加

### 画像をアップロードする
1. 画像フィールドをクリック
2. 「Upload」タブを選択
3. ファイルを選択してアップロード
4. 画像が`/images/`フォルダに保存される

### 既存の画像を選択する
1. 画像フィールドをクリック
2. 「Choose an image」から既存の画像を選択

## 8. トラブルシューティング

### 変更が反映されない
- GitHub Pagesの更新には1〜5分程度かかります
- ブラウザのキャッシュをクリアしてみてください（Ctrl + F5）

### ログインできない
- GitHubアカウントの認証を確認
- リポジトリへのアクセス権限を確認

### 画像が表示されない
- 画像のパスが正しいか確認
- 画像ファイルが`/images/`フォルダにあるか確認

### バナーが表示されない
- 「表示する」のチェックが入っているか確認
- 画像がアップロードされているか確認
- ファイル形式が正しいか確認（JPG、PNG、GIF推奨）

### 色が変わらない
- HEX形式が正しいか確認（例: `#ff0000`）
- 保存とPublishを両方実行したか確認
- ブラウザのキャッシュをクリア

## 11. 注意事項

✅ **改行について**: テキスト内で改行したい場合は、エンターキーで改行を入れてください。サイト上でも改行として表示されます。

✅ **文字数制限**: 基本的に文字数制限はありませんが、極端に長い文章は読みにくくなる可能性があります。

✅ **HTML編集**: 本システムでは、HTML構造の編集は管理画面からできません。デザインやレイアウトの大幅な変更が必要な場合は、開発者にご相談ください。

✅ **バックアップ**: GitHubに変更履歴が自動的に保存されるため、過去の状態に戻すことが可能です。

✅ **HEX形式**: カラーコードは必ず`#`で始まり、6桁の英数字（0-9、A-F）で構成されます。

✅ **画像サイズ**: 大きすぎる画像はページの表示速度に影響します。適切なサイズに圧縮してからアップロードしてください。
