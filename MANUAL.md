# 有限会社Rispondere コーポレートサイト 管理画面編集マニュアル

## 1. 管理画面へのアクセス

### 管理画面URL
```
https://[あなたのドメイン]/admin/
```

### ログイン方法
1. 管理画面URLにアクセス
2. GitHubアカウントでログイン
3. リポジトリへのアクセス権限を承認

## 3. 基本的な編集手順

### ステップ1: ログイン
1. 管理画面URL（`/admin/`）にアクセス
2. 「Login with GitHub」をクリック
3. GitHubアカウントでログイン

### ステップ2: 編集したい項目を選択
1. 左サイドバーから編集したいカテゴリを選択
   - 「サイト設定」
   - 「ページ」
   - 「よくある質問」
2. 編集したい具体的な項目をクリック

### ステップ3: 内容を編集
1. テキストフィールドに新しい内容を入力
2. 改行を含む場合は、エンターキーで改行を入れる
3. 画像を変更する場合は、画像フィールドから選択またはアップロード

### ステップ4: 保存
1. 編集が完了したら、右上の「Save」ボタンをクリック
2. 「Publish」を選択して公開

### ステップ5: 確認
1. 数分後、実際のサイトにアクセスして変更を確認
2. GitHub Pagesの更新には1〜5分程度かかる場合があります

---

## 4. よくある質問（FAQ）の編集

### 新しいFAQを追加する
1. 左サイドバーから「よくある質問」を選択
2. 右上の「New よくある質問」をクリック
3. 質問と回答を入力
4. 表示順（数字が小さいほど上に表示）を設定
5. 「Save」→「Publish」で公開

### 既存のFAQを編集する
1. 左サイドバーから「よくある質問」を選択
2. 編集したいFAQをクリック
3. 内容を編集
4. 「Save」→「Publish」で公開

### FAQを削除する
1. 編集画面で「Delete entry」をクリック
2. 確認メッセージで「OK」を選択

---

## 5. 画像の編集・追加

### 画像をアップロードする
1. 画像フィールドをクリック
2. 「Upload」タブを選択
3. ファイルを選択してアップロード
4. 画像が`/images/`フォルダに保存される

### 既存の画像を選択する
1. 画像フィールドをクリック
2. 「Choose an image」から既存の画像を選択

---

## 6. 直接編集する方法（代替方法）

管理画面を使わずに、GitHubから直接編集することも可能です。

### GitHubでの編集手順
1. GitHubリポジトリにアクセス
2. `/content/`フォルダ内のJSONファイルを開く
3. 鉛筆アイコン（Edit）をクリック
4. JSONファイルの内容を編集
5. 「Commit changes」で保存

**注意**: JSON形式を崩さないように注意してください。

---

## 7. トラブルシューティング

### 変更が反映されない
- GitHub Pagesの更新には1〜5分程度かかります
- ブラウザのキャッシュをクリアしてみてください（Ctrl + F5）

### ログインできない
- GitHubアカウントの認証を確認
- リポジトリへのアクセス権限を確認

### 画像が表示されない
- 画像のパスが正しいか確認
- 画像ファイルが`/images/`フォルダにあるか確認

## 9. 注意事項

✅ **改行について**: テキスト内で改行したい場合は、エンターキーで改行を入れてください。サイト上でも改行として表示されます。

✅ **文字数制限**: 基本的に文字数制限はありませんが、極端に長い文章は読みにくくなる可能性があります。

✅ **HTML編集**: 本システムでは、HTML構造の編集は管理画面からできません。デザインやレイアウトの大幅な変更が必要な場合は、開発者にご相談ください。

✅ **バックアップ**: GitHubに変更履歴が自動的に保存されるため、過去の状態に戻すことが可能です。
