## 💾 容量の目安

### サイズ別容量目安

| 画像サイズ | 適正容量 | 最大容量 | 圧縮必須 |
|----------|---------|---------|---------|
| 〜600×400px | 50〜150KB | 300KB | ✅ |
| 600×400〜1200×800px | 150〜300KB | 500KB | ✅ |
| 1200×800px以上 | 300〜500KB | 1MB | ✅必須 |

### ⚠️ 容量オーバーの目安

| 容量 | 状態 | 対処 |
|------|------|------|
| 〜500KB | ✅ 理想的 | そのまま使用 |
| 500KB〜1MB | ⚠️ やや大きい | できれば圧縮 |
| 1MB〜3MB | 🚨 大きすぎ | **必ず圧縮** |
| 3MB以上 | ❌ 使用不可 | **圧縮必須** |

## 🔧 画像圧縮ツール（無料）

### おすすめツールTOP3

#### 1位: TinyPNG/TinyJPG ⭐⭐⭐⭐⭐
- **URL**: https://tinypng.com/
- **特徴**: 最大70%圧縮、品質劣化少ない
- **使い方**: ドラッグ&ドロップ → ダウンロード
- **対応**: PNG、JPG（最大20枚まで一括）

#### 2位: Squoosh ⭐⭐⭐⭐
- **URL**: https://squoosh.app/
- **特徴**: Google製、リアルタイムプレビュー
- **使い方**: 画像をドロップ → 圧縮率調整 → ダウンロード
- **対応**: JPG、PNG、WebP、AVIF

#### 3位: JPEG-Optimizer ⭐⭐⭐
- **URL**: http://jpeg-optimizer.com/
- **特徴**: 圧縮率を細かく設定可能
- **使い方**: アップロード → 品質選択 → 最適化
- **対応**: JPG専用

## 🎨 画像形式の使い分け

### 簡単チャート

```
画像を準備 → 何の画像？

┌─ 写真・風景
│   → JPG形式
│   → 容量: 200〜500KB
│   → 圧縮: TinyJPG
│
┌─ ロゴ・シンプルなデザイン
│   → SVG形式（最優先）
│   → 容量: 5〜20KB
│   → 拡大縮小自由
│
└─ ロゴ・透過が必要
    → PNG形式
    → 容量: 50〜200KB
    → 圧縮: TinyPNG
```

### 形式別の特徴

| 形式 | 最適な用途 | メリット | デメリット | 圧縮 |
|------|----------|---------|-----------|------|
| **JPG** | 写真・風景 | 容量小 | 透過不可 | 必須 |
| **PNG** | ロゴ・透過画像 | 高品質・透過OK | 容量大 | 推奨 |
| **SVG** | ロゴ・アイコン | 超軽量・拡大自由 | 複雑な画像不可 | 不要 |
| **WebP** | 次世代形式 | JPGより30%軽量 | 古いブラウザ非対応 | 不要 |

## 💡 よくある質問

### Q1. どのサイズで作ればいい？
**A**: 迷ったら**1920×600px（横長）** または **400×800px（縦長）** を使用してください。

### Q2. 容量はどこで確認できる？
**A**: ファイルを右クリック → プロパティ（Windows）または情報を見る（Mac）

### Q3. 圧縮したら画質が落ちない？
**A**: TinyPNGなら品質劣化はほとんどありません（肉眼では判別不可）。

### Q4. SVGって何？
**A**: ベクター形式の画像。拡大縮小しても綺麗で、容量も超軽量。ロゴに最適。

### Q5. スマホの写真はそのまま使える？
**A**: サイズが大きすぎるため、必ずリサイズ + 圧縮してください。
