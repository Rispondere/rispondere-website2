# ロゴ画像について

## ファイル名
`rispondere-logo.png`

## 推奨サイズ
- 横幅: 150-200px
- 高さ: 40-50px
- 形式: PNG（透過背景推奨）

## 現在の状態
現在はプレースホルダーとして、お客様提供のロゴ画像を配置してください。

## 差し替え方法

### 1. 管理画面から（推奨）
1. `/admin/` にアクセス
2. 「サイト設定」→「ロゴ画像」をアップロード
3. 自動的に `/images/logo/rispondere-logo.png` として保存されます

### 2. 手動で差し替え
1. ロゴ画像ファイルを用意
2. ファイル名を `rispondere-logo.png` に変更
3. `/images/logo/` フォルダに上書き保存

## 注意事項
- ファイル名は変更しないでください（HTMLから参照されています）
- 推奨サイズを守ることで、レイアウトが崩れません
- PNG形式で透過背景にすると、どの背景色でも美しく表示されます
